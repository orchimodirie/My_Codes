##  Repository Structure

The codebase is organized into three core directories:

### 1. `Courses/`
* **Description:** Contains official academic laboratory exercises, machine problems, and projects assigned during university courses.
* **Focus:** Course-specific syllabi implementations, structural assignments, and academic milestones.

### 2. `Practice/`
* **Description:** Dedicated space for standalone coding challenges, algorithmic problem-solving (e.g., LeetCode, HackerRank), and syntax experimentation.
* **Focus:** Problem-solving logic, optimization, and refining competitive programming skills.

### 3. `Topics/`
* **Description:** Structured notes and code snippets categorized by specific computer science paradigms and core programming concepts.
* **Focus:** Object-Oriented Programming (OOP), Data Structures and Algorithms (DSA), memory management, and language-specific deep dives.
